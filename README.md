# Skill Distillation — 认知蒸馏技能库

用**仓颉·认知植入术**从历史人物与经典文本中蒸馏出思维方式，而非仅仅提取"他说了什么"——而是提取"他为什么这样想"。

## 这是什么

每一个 Skill 都是一个完整的认知上下文植入包，包含：

- **信念及其来源** — 他相信什么，以及这些信念从哪些具体经历中长出来
- **关键决策** — 他在哪些岔路口做了什么选择，为什么
- **栽过的跟头** — 他承认的错误和失败，以及从中学到了什么
- **内在矛盾** — 他自己没有完全解决的认知张力
- **明确不懂的** — 他的认知边界在哪里

## 已有技能

### 🌿 李时珍（lishizhen-perspective）

> 「身如逆流船，心比铁石坚。」

三次科举落第，27年重修本草，亲尝毒药验证古籍，向渔夫樵夫求教——一个用实证对抗千年谬误的药圣。

**适用场景：** 药物考证、分类体系设计、实证方法论、传统知识批判性继承、药食同源

**核心认知特征：**
- 亲验胜于古籍——不验证的药物知识可以杀人
- 贱者之言亦可贵——民间从业者才是活的知识库
- 分类须循自然之序——推翻千年三品分类法，创立16部60类体系
- 实录精神 vs 伦理批判的内在挣扎

### ☯️ 黄帝内经（huangdineijing-perspective）

> 「阴阳者，天地之道也，万物之纲纪，变化之父母。」

战国至西汉数代医家的集体智慧，在无法打开活人身体的时代，学会了从外在之象推知内在之藏。

**适用场景：** 中医整体观、取象比类、天人合一、治未病、身心关系、预防vs治疗

**核心认知特征：**
- 天人合一——脱离自然环境谈健康，是无根之木
- 治未病——最高明的医术是让人不生病
- 取象比类——黑箱前唯一的认知通路
- 素问重推演 vs 灵枢重实证的未解张力

### 🎨 图片提示词生成器（image-prompt-generator）

一个交互式 GPT Image / image2 提示词生成技能。先引导用户选择图片类型，再按模板收集关键信息，最后生成可复制的高质量图片提示词，并可询问是否调用 `gpt-image-2` 直接生图。

**适用场景：** 小红书封面、概念海报、城市旅行海报、清透人像、动漫角色杂志封面、产品视觉、GPT Image 2 进阶工作流

**核心能力：**
- 交互式图片类型选择与模板化信息收集
- LUMEN ATLAS 城市海报、清透人像、海贼王人物海报等风格模板
- GPT Image 2 image-only 工作流边界说明
- 输出视觉策略、完整 prompt、生成设置和 QA 检查清单

## 如何使用

将 `SKILL.md` 的内容作为系统提示（System Prompt）注入 AI 对话，即可让 AI 以该人物的认知方式思考问题。

每个 SKILL.md 的 frontmatter 中包含触发条件（`description` 字段），可用于自动匹配场景。

## 蒸馏方法

本项目使用**仓颉·认知植入术**进行提取，核心原则：

1. **不提取"他说了什么"，提取"他为什么这样想"** — 信念必须有形成故事
2. **必须有具体场景** — 每条认知都锚定在真实决策或失败教训上
3. **必须暴露矛盾** — 内在张力比一致性更有认知价值
4. **必须标注边界** — "明确不懂的"比"懂的"更能防止幻觉

## 素材可信度

每个技能的 `references/research/01-sources.md` 记录了完整的素材来源清单，标注了一手/二手、可信度等级，并做了交叉验证。蒸馏过程中区分了：
- 一手来源（亲撰文本、原文）
- 二手权威来源（学术专著、专业考据）
- 二手网络来源（需交叉验证）
