# Intake Guide

Use this file after image type selection. Ask only for missing required fields. If the user already supplied enough detail, infer optional fields and draft immediately.

## XHS Cover

Required:
- Topic or title.
- Target audience.
- Cover family: Minimal Impact, Emotional Value, Knowledge Card, or Product Seeding.

Optional:
- Mood, product, visible text, color preference, aspect ratio, and forbidden elements.

Ask:
"这张小红书封面更偏知识卡、情绪价值、极简冲击，还是产品种草？标题/主题是什么？"

## Concept Poster

Required:
- Core word, phrase, concept, or visible text.
- Desired poster family when named: Typography Aesthetic, Advanced Concept, Object Metaphor, Spatial Tension, or LUMEN ATLAS City Poster.

Optional:
- Mood, metaphor direction, visual object, palette, aspect ratio, and forbidden cliches.

Ask:
"核心文字或概念是什么？你更想要字体美学、视觉隐喻、物件隐喻、空间张力，还是城市地图集海报？"

## City Atlas Poster

Required:
- City name.

Optional:
- Poetic subtitle, coordinates, landmarks, mood, sky/light condition, and Chinese editorial lines.

Ask:
"城市名是什么？是否有指定地标、经纬度、诗意副标题或光线氛围？"

## Epic Narrative Poster

Required:
- Story subject, character, IP, novel, or theme.
- Main silhouette or container image idea.

Optional:
- Internal scenes, symbols, supporting characters, setting, mood, and aspect ratio.

Ask:
"主角或主题是什么？剪影内部要放哪些场景、人物关系、地标或象征物？"

## Portrait Editorial

Required:
- Person or subject type.
- Portrait family: Magazine Editorial, Lifestyle, Clear Luminous, or Cinematic.

Optional:
- Pose, expression, wardrobe, hair/makeup, background, lighting, palette, and ratio.

Ask:
"人物是谁或是什么类型？想要杂志编辑、生活感、清透明亮，还是电影感写真？"

## Anime Character Fashion Cover

Required:
- Character name.
- Style variant: Mediterranean Resort Muse, Intellectual Coastal Palace, or Royal Crimson Empress.

Optional:
- Masthead text, wardrobe, pose, character traits, and negative constraints.

Ask:
"角色名是什么？三种风格选哪一个：地中海度假风、优雅知性风、女王奢华风？"

## Product Visual/Edit

Required:
- Product or source image description.
- Operation: refinement, exploded view, compositing, light effect, industrial line art, or replacement.
- What must be preserved and what must change.

Optional:
- Scene, background, material emphasis, camera angle, output platform, and ratio.

Ask:
"这是产品精修、换背景/合成、结构展示、光效、线稿，还是产品替换？哪些外形、材质、品牌信息必须保留？"

## GPT Image 2 Advanced Workflows

Required:
- Workflow type: Layered Poster, Batch Variants, Life Milestone Narrative, Fitness Infographic, Photo Doodle Annotation, or Palm Reading Entertainment.
- Source image requirement when needed.
- Output count for batch tasks.

Optional:
- Safety/privacy boundaries, variant axes, layer names, labels, visible text, and ratio.

Ask:
"你要做分层图、批量生图、人生照片、健身图解、照片手绘标注，还是手相娱乐图？是否已经有需要上传的源图？"

Rules:
- Apply the Image Model Only Gate from `SKILL.md`.
- For fitness, include non-medical framing.
- For palm-reading entertainment, include entertainment-only and no-scientific-basis framing.
- If source images are required but absent, produce a prompt template only or ask for the image.
