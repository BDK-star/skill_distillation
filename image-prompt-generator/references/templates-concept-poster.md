# Concept Poster Templates

Use these templates for 概念海报, 字体美学, visual metaphor posters, art posters, and symbolic brand visuals.

## Shared Variables

- `{core_text}`: exact visible word, phrase, short sentence, or letter group
- `{concept}`: theme or message
- `{language}`: Chinese, English, mixed, or other
- `{metaphor}`: non-cliche metaphor
- `{visual_elements}`: 1-3 primary symbolic elements
- `{typography}`: type treatment
- `{composition}`: layout and spatial logic
- `{aspect_ratio}`: selected ratio and why it serves the concept
- `{palette}`: color palette
- `{lighting}`: light logic
- `{material}`: material or graphic language
- `{texture}`: print, paper, grain, ink, or surface texture
- `{mood}`: emotional direction
- `{negative_constraints}`: user-specific avoid list
- `{city_name}`: target city name for city atlas posters
- `{city_poetic_name}`: short poetic subtitle or city nickname
- `{coordinates}`: real or approximate geographic coordinates when known
- `{landmarks}`: skyline, waterfront, architecture, mountains, bridges, landmarks, or local geographic features

## Template: Typography Aesthetic Master Poster

Use when the visible word itself must be the main visual subject. This is the strongest template for "字体美学", word-based concept images, and custom typographic posters.

Prompt skeleton:

```md
Create a top-tier typography aesthetic concept image centered on "{core_text}".

Core principle:
The visible text "{core_text}" must be the largest, clearest, most memorable subject in the image. It should be seen and understood at first glance. Every other visual element must support, explain, frame, compress, release, contrast with, or amplify the text. Do not let background, people, objects, light, or decoration overpower the text.

Semantic direction:
First interpret the word's literal meaning, emotional temperament, cultural associations, psychological tension, and deeper metaphor. Use this non-cliche metaphor: {metaphor}. The final image should feel like this word's visual thought, not just an illustration of the word.

Typography:
{typography}. The text itself should express the meaning through stroke proportion, compression, stretch, fracture, growth, negative space, material, spatial force, or interaction with symbols. If the text is Chinese, render each character accurately with correct structure and no fake characters. If the text is English, spell it accurately with refined spacing, weight, baseline, rhythm, and negative space.

Visual system:
Use only a few precise supporting elements: {visual_elements}. They should interact with the text through growth from strokes, cutting through letters, enclosing, supporting, compressing, releasing, casting symbolic shadows, becoming a container, or forming a second meaning through negative space.

Composition and format:
{composition}. Aspect ratio: {aspect_ratio}. The ratio itself should strengthen the word's spatial tendency: square for concentration or order, vertical for pressure or ascent, horizontal for flow or distance, ultra-wide for epic systems, ultra-tall for depth or falling.

Style:
Choose a style derived from the word, not a fixed template. Suitable directions may include refined minimal design, constructivist poster, Swiss grid, contemporary Japanese graphic design, modern calligraphy, luxury editorial visual, futurist precision, classical inscription, surreal concept poster, experimental typography, museum-poster design, or sculptural type installation.

Color, light, material:
Palette: {palette}. Lighting: {lighting}. Material/graphic language: {material}. Texture: {texture}. All choices must support the text and metaphor.

Avoid:
Tiny text, background stealing attention, default fonts, misspelled or fake text, extra explanatory text, random symbols, irrelevant decorations, repeated one-style templates, cheap 3D text, vulgar neon, PPT-cover look, Canva template look, e-commerce ad style, childish illustration, blurry low-resolution edges, generic AI randomness, watermarks, logos, QR codes, and {negative_constraints}.
```

## Template: Advanced Concept Poster

Use when image and text should form a minimal, clever, memorable visual sentence.

Prompt skeleton:

```md
Create a high-end graphic concept poster based on "{core_text}".

Concept analysis:
{concept}. Emotional direction: {mood}. Core metaphor: {metaphor}. The poster should make viewers feel that the image and this word are strongly bound together.

Composition:
The visible text "{core_text}" is the first visual layer, large and readable. The image narrative is the second layer. Any small auxiliary information is the third layer and should be used only if necessary. Use a clean, powerful composition with intelligent negative space. Image elements should relate to the text by standing before it, embedding into it, cutting it, using its negative space, or forming contrast with it.

Visual expression:
Use {visual_elements} to express the concept through relationship, scale contrast, direction, distance, occlusion, action moment, order, conflict, absurdity, or poetic restraint. Prefer one condensed strong image over many explanatory objects.

Style:
Refined graphic-art poster with flat design strength, subtle collage, silkscreen, lithograph, or print texture. Keep the color system clean and limited: {palette}. Texture: {texture}. Avoid dirty or low-level collage.

Typography:
Text must feel born from the image rather than pasted on. If Chinese, keep characters correct and central. If English, preserve spelling and give the type strong shape and visual force. Optional small subtitle only when it deepens the theme.

Avoid:
Cheap templates, advertising-page look, ordinary e-commerce layout, over-explanation, crowded elements, generic beauty, fake characters, unreadable type, and {negative_constraints}.
```

## Template: LUMEN ATLAS City Poster

Use when the user wants a National Geographic-inspired city poster, city atlas editorial cover, luxury travel magazine poster, or a reusable city-name replacement template. This template works best when the output prompt is written in English while preserving any required Chinese visible text exactly.

Prompt skeleton:

```md
Create a premium vertical city atlas editorial poster for {city_name}, titled "LUMEN ATLAS".

Overall direction:
The poster should feel like a luxury travel magazine cover, combining National Geographic-inspired field-documentary atmosphere with Apple-like minimal layout and Swiss editorial typography. The design is refined, spacious, cinematic, and museum-quality, not a tourist advertisement.

Top atlas system:
At the top, set the title "LUMEN ATLAS" in elegant modern typography. Behind or around it, place large translucent geographic coordinates for {city_name}: {coordinates}. Use subtle longitude and latitude arcs, cartographic coordinate markers, topographic contour lines, river paths, navigation indicators, route markers, and minimal atlas-inspired graphics. Do not use clock scales, horoscope graphics, decorative compasses, or fake UI widgets.

City identity:
Main city title: "{city_name}".
Poetic subtitle: "{city_poetic_name}".
Use the city name as the primary local identity, with the subtitle as a quieter editorial accent. If Chinese text is used, keep it short, large enough to read, and accurately written.

Central image:
Show an atmospheric cinematic view of {city_name}, emphasizing {landmarks}. The city should occupy roughly 30% of the poster and appear elegant, distant, immersive, and uncrowded. The sky and atmosphere should occupy roughly 70%, using extraordinary natural light such as golden hour, blue hour, sunrise glow, sunset haze, storm light, mist, or twilight color.

Editorial text:
In the center or lower-middle, include three short poetic Chinese editorial lines describing the city's mood, light, architecture, geography, and atmosphere. The text should feel like magazine copy, not a slogan wall. Keep the lines readable and integrated into the layout.

Bottom navigation:
At the bottom, replace any ordinary timeline with an elegant atlas navigation system: landmark nodes, coordinate points, route markers, small geographic indicators, and subtle map-line structure. Keep it quiet, precise, and premium.

Visual style:
Photographic realism with fine-art travel photography quality. Luxury Swiss editorial design, minimalist information graphics, sophisticated typography hierarchy, large negative space, atmospheric color grading, ultra-clean layout, refined geometric alignment, high-end branding poster, soft glow, cinematic depth, vertical poster format, 4K ultra detailed.

Avoid:
Crowded tourist collage, postcard kitsch, stock travel ad style, random landmark pile-up, fake coordinates, unreadable microtext, fake Chinese characters, heavy yellow border imitation, overdecorated map icons, clock-scale graphics, cheap Canva layout, excessive gradients, watermarks, logos, QR codes, and {negative_constraints}.
```

## Template: Object Metaphor Poster

Use when one object can carry the theme through an unexpected relationship.

Prompt skeleton:

```md
Create a concept poster for "{concept}".

Core visible text: "{core_text}".
Metaphor:
{metaphor}

Composition:
Use one symbolic object as the main visual anchor: {visual_elements}. The object should express the theme through scale, material, position, transformation, or interaction with the text. Keep the layout minimal and memorable. Text remains large, readable, and part of the designed composition.

Style:
High-end editorial poster design, {palette} palette, {texture}, clean negative space, strong visual hierarchy, and precise art direction. Mood: {mood}.

Avoid:
Over-explaining the idea, adding too many objects, generic surrealism, fake text, stock-image look, cheap collage, and {negative_constraints}.
```

## Template: Spatial Tension Poster

Use when the concept is best expressed through space: pressure, distance, enclosure, expansion, falling, fracture, or silence.

Prompt skeleton:

```md
Create a visual metaphor poster for "{concept}".

Visible text: "{core_text}".
Spatial metaphor:
{metaphor}

Composition:
Use space itself as the main expressive force: {composition}. The image should show tension through distance, compression, blankness, scale imbalance, enclosure, rupture, or release. Use very few visual elements: {visual_elements}. The text should be large and readable, positioned as part of the spatial relationship.

Style:
Restrained graphic poster, intelligent negative space, controlled {palette} palette, subtle {texture}, and a refined museum-poster quality. Mood: {mood}.

Avoid:
Literal cliches, crowded layouts, decorative noise, fake characters, unreadable text, random symbols, watermarks, and {negative_constraints}.
```
